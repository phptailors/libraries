3.7. Multiple context arguments

Multiple arguments may be passed to with():

1with($cm1, $cm2, ...)(function ($arg1, $arg2, ...) {
2   # body of the user-provided callback ...
3});

For every value $cm1, $cm2, …, passed to with() a corresponding value is passed as an argument to the user-provided callback. Assuming, $cm1, $cm2, …, are Context Managers, the corresponding arguments of the user-provided callback will receive

  • $arg1 = $cm1->enterContext(),

  • $arg2 = $cm2->enterContext(),

  • …

The context managers cm1, cm2, …, are invoked in the same order as they appear on the argument list to with() when entering the context (enterContext()) and in the reverse order when exiting the context (exitContext()).

Let’s use the following simple context manager to illustrate this

 1final class MyInt implements ContextManagerInterface
 2{
 3    public int $value;
 4
 5    public function __construct(int $value)
 6    {
 7        $this->value = $value;
 8    }
 9
10    public function enterContext(): int
11    {
12        echo "enter: " . $this->value . "\n";
13        return $this->value;
14    }
15
16    public function exitContext(?\Throwable $exception = null) : bool
17    {
18        echo "exit: " . $this->value . "\n";
19        return false;
20    }
21}

The order of argument processing may be illustrated by the following test

1with(new MyInt(1), new MyInt(2), new MyInt(3))(function (int ...$args) {
2    echo '$args: ' . implode(', ', $args) . "\n";
3});

The output from above snippet will be

1enter: 1
2enter: 2
3enter: 3
4$args: 1, 2, 3
5exit: 3
6exit: 2
7exit: 1
