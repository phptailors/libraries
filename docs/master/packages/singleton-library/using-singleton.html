2.3. Using Singleton

Just use SingletonTrait in your class and the class becomes a singleton. It is also recommended to annotate the class that it implements SingletonInterface. The following snippet provides a complete example of trivial singleton, that is an empty class that is a singleton.

Trivial Singleton example
 1<?php
 2
 3use Tailors\Lib\Singleton\SingletonInterface;
 4use Tailors\Lib\Singleton\SingletonTrait;
 5
 6final class TrivialSingleton implements SingletonInterface
 7{
 8    use SingletonTrait;
 9}
10
11$object = TrivialSingleton::getInstance();
12
13// vim: syntax=php sw=4 ts=4 et:

The SingletonTrait provides features that shall appear in any typical singleton:

  • public static getInstance() method returning the single instance,

  • private __construct() method,

  • private __clone() method,

  • public __wakeup() method always throws SingletonException.

In addition:

  • a protected method initializeSingleton() is invoked from the private constructor when the single instance gets created; the class that includes the SingletonTrait may override the method to initialize the single instance on its own. The default implementation is just an empty method.

The complete functionality of the SingletonTrait may be illustrated with the following example

Usage of Singleton
 1<?php
 2
 3use Tailors\Lib\Singleton\SingletonInterface;
 4use Tailors\Lib\Singleton\SingletonTrait;
 5
 6final class CountSingleton implements SingletonInterface
 7{
 8    use SingletonTrait;
 9
10    static public $count = 0;
11
12    public function initializeSingleton()
13    {
14        self::$count += 1;
15        print("initializeSingleton(): count: ".self::$count."\n");
16    }
17}
18
19print("main 1: count: ".CountSingleton::$count."\n");
20$var1 = CountSingleton::getInstance();
21print("main 2: count: ".CountSingleton::$count."\n");
22$var2 = CountSingleton::getInstance();
23print("main 3: count: ".CountSingleton::$count."\n");
24print("main 4: (var1 === var2): ".($var1 === $var2 ? 'true' : 'false')."\n");
25
26// vim: syntax=php sw=4 ts=4 et:

1main 1: count: 0
2initializeSingleton(): count: 1
3main 2: count: 1
4main 3: count: 1
5main 4: (var1 === var2): true
