*******************************
Installing PhpTailors Libraries
*******************************

Use composer_ to install the framework or some of its components.

To install whole PhpTailors Libraries for your project, type

.. code-block:: shell

      php composer.phar require "phptailors/libraries:dev-master"

If you only need a component, for example ``phptailors/ldaplib-library``, then

.. code-block:: shell

      php composer.phar require "phptailors/ldaplib-library:dev-master"

.. note::

   We have not released PhpTailors Libraries yet, so it's only installable from
   gitlab_.

.. _composer: https://getcomposer.org/
.. _gitlab: https://github.com/phptailors/libraries/
