bool exitContext(Throwable|null $exception = null)

Exit the runtime context and return a Boolean flag indicating if any exception that occured should be suppressed.

If an exception occurred while executing the user function, the $exception argument contain the exception. Otherwise it contains null.

Returning a true from this method will cause the "with" caller to suppress the exception and continue execution.
