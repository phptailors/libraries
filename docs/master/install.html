1. Installing PhpTailors Libraries

Use composer to install the framework or some of its components.

To install whole PhpTailors Libraries for your project, type

1php composer.phar require "phptailors/libraries:dev-master"

If you only need a component, for example phptailors/ldaplib-library, then

1php composer.phar require "phptailors/ldaplib-library:dev-master"

Note

We have not released PhpTailors Libraries yet, so it’s only installable from gitlab.
